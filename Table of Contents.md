# Table of Contents

This documentation contains a series of hands-on laboratories covering Microsoft Fabric, Azure Data Services, Snowflake, Apache Kafka, and real-time data processing technologies.

| Lab        | Title                                                       | Description                                                                                       |
| ---------- | ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| **LAB 1**  | **Data Exploration & Transformation in Microsoft Fabric**   | Learn how to explore, clean, and transform datasets using Microsoft Fabric.                       |
| **LAB 2**  | **Snowflake — Cloud Data Warehousing**                      | Explore cloud-native data warehousing concepts and implementation using Snowflake.                |
| **LAB 3**  | **Configuring Single-Node Kafka Cluster**                   | Configure and manage a single-node Apache Kafka cluster for development and testing environments. |
| **LAB 4**  | **OneLake — Unified Storage in Microsoft Fabric**           | Understand Microsoft Fabric OneLake architecture and unified storage capabilities.                |
| **LAB 5**  | **Fabric Data Factory — Pipeline Design & Automation**      | Design, build, and automate data integration pipelines using Fabric Data Factory.                 |
| **LAB 6**  | **Transform & Load into Data Warehouse using Spark**        | Transform datasets with Apache Spark and load them into a data warehouse.                         |
| **LAB 7**  | **Ingest & Load Data into the Data Warehouse**              | Perform data ingestion and loading processes into a centralized data warehouse.                   |
| **LAB 8**  | **Transform Data with Azure Data Factory**                  | Build transformation workflows using Azure Data Factory.                                          |
| **LAB 9**  | **Real-Time Stream Processing with Azure Stream Analytics** | Process streaming data in real time using Azure Stream Analytics.                                 |
| **LAB 10** | **Stream Processing with Event Hub + Databricks**           | Implement real-time stream processing using Azure Event Hubs and Azure Databricks.                |
